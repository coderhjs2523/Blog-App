# 📝 BlogNest

A modern full-stack blogging platform built with **React**, **Appwrite**, and **TinyMCE**. Users can create, edit, publish, and manage blogs with a clean and responsive interface.

## 🚀 Live Demo

https://blog-app-eosin-six.vercel.app/

## 📸 Features

- 🔐 User Authentication (Login & Signup)
- ✍️ Rich Text Editor (TinyMCE)
- 📝 Create, Update, and Delete Posts
- 📷 Image Upload Support
- 📖 Read Published Blogs
- 🔒 Protected Routes
- 🎨 Responsive UI
- ⚡ Fast Performance with Vite
- ☁️ Appwrite Backend Integration

## 🛠️ Tech Stack

### Frontend
- React.js
- React Router DOM
- Redux Toolkit

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/your-username/blog-app.git
```

Go to project directory

```bash
cd blog-app
```

Install dependencies

```bash
npm install
```

Create a `.env` file and add your Appwrite credentials

```env
VITE_APPWRITE_URL=
VITE_APPWRITE_PROJECT_ID=
VITE_APPWRITE_DATABASE_ID=
VITE_APPWRITE_COLLECTION_ID=
VITE_APPWRITE_BUCKET_ID=
VITE_TINYMCE_API_KEY=
```

Run the project

```bash
npm run dev
```
